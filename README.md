# Brackets

# React + TypeScript + Vite + Mobx
The Brackets project is a webapp that can be used to create karate tournament brackets. 

Try it here: [https://nikojo.github.io/Brackets](https://nikojo.github.io/Brackets).


# Building

Get modules
```
npm install
```

Run the vite dev environment
```
npm run dev
```

Or build it
```
npm run build
```

And run it
```
npm run preview
```


# Github pages
This project is integrated with Github pages. Once changes are pushed, the project is built and deployed to [https://nikojo.github.io/Brackets](https://nikojo.github.io/Brackets).
